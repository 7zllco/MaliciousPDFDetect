This PDF parser is free, simple online application that allows you to detect malicious code in your PDF file. Ewha Womans University students have decided to publish a free online tool that uses machine learning algorithm to parse data from multi PDf files. The tool works is in your bowser and nothing is stored on the server. Therefore, our tool is safe to use even for files with sensitive information.
